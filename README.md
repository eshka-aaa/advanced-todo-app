\# 📝 Advanced ToDo App



A modern and responsive ToDo application built with Vanilla JavaScript.



\## 🚀 Features



\* ✅ Add tasks

\* ✏️ Edit tasks

\* 🗑 Delete tasks

\* ✔️ Mark tasks as completed

\* ⭐ Pin important tasks

\* 🔍 Search tasks

\* 📂 Task categories

\* 🚨 Task priorities

\* 📅 Deadlines

\* 📊 Progress bar

\* 🌙 Dark mode

\* 🌍 Language switcher (RU / EN)

\* 🔔 Toast notifications

\* 💾 localStorage support

\* 📱 Responsive design

\* ✨ Smooth animations



\---



\## 🛠 Technologies Used



\* HTML5

\* CSS3

\* JavaScript (Vanilla JS)

\* localStorage API
